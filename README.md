## 01. process

1. top - display tasks

## 02. each info

### TOP : 시스템 프로세스/메모리 사용 현황을 실시간으로 출력한다.

사용법 : &top[옵션]

옵션|의미|
---|---|
-n|지정한 숫자만큼 화면 출력을 갱신|
-u|지정한 사용자의 프로세스를 모니터링|
-b|출력결과를 파일이나 다른 프로그램으로 전달|
-d|화면갱신주기를 초 단위로 설정|
-p|지정한 PID 프로세스를 모니터링
